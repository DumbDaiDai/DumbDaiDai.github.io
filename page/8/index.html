无题·天河分明晦

发表于 2024-09-07 更新于 2024-10-24 分类于 仿古诗歌 本文字数： 48 阅读时长 ≈ 1 分钟

天河分明晦，舟火戴星辉。
浮月饮风醉，颓云抱霜归。
有处寻欢喜，无闲事伤悲。
花间本无趣，何必更贪杯。

无题·黑云漫卷兼落日

黑云漫卷兼落日，雨色疏疏风声疲。
阶上老桑着黄叶，堂前新竹抽青枝。
只言春来花开早，不道冬伫寒去迟。
自古万岁多如荠，千载谁复堪此期。

万难歌

发表于 2024-09-07 更新于 2024-10-24 分类于 仿古诗歌 本文字数： 183 阅读时长 ≈ 1 分钟

庸夫与道，三江难渡：惧愤难破，情思难绝，执欲难灭。
一叶扁舟难觅，如隔沧海难济，疾浪难平，迷烟难去。举首秋水难断，高风难追，明月难掇。
一回长梦难醒，似煎文火难静，浮心难安，残念难拾。低眉苦恨难除，霰泪难干，殷忧难尽。
手无金樽难酹，无琵琶难诉，无洞箫难托，百般难咽。雾里秋花难一见，花落千里路难行。几多乱麻难数，短须难梳。
破败孤帆难支，摧楫难执，雨下残烛难炙，归期难知。

无题·不知何方草色青

发表于 2024-09-07 更新于 2024-10-24 分类于 仿古诗歌 本文字数： 64 阅读时长 ≈ 1 分钟

不知何方草色青，徒留数般过客影。
有闲长望江边雨，无心细数池上萍。
胡言镜中风花雪，乱语井下日月星。
尽洒一壶冰心水，莫问云间几时晴。
